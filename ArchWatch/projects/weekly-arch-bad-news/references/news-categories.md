# 建筑行业负面新闻分类框架

---

# 模式A：行业周报

## 时效性要求（强制）
- **时间限制**：只收录发布时间在当前日期前7天内的新闻
- **时间计算**：当天为第0天，往前推7天（共8天范围）
- **示例**：若今天是2026年3月2日，则只收录2月23日至3月2日发布的新闻
- **超过7天的新闻一律不收录，即使事件重要**

# 模式B：单位查询

## 触发条件
用户消息以"帮我查询"开头，提取单位名称进行查询：
```
示例：
- "帮我查询深圳市建筑设计研究总院" → 单位名称：深圳市建筑设计研究总院
- "帮我查询中建三局" → 单位名称：中建三局
```

## 时间范围
- **不限制时间**：可检索历史新闻
- **数量限制**：最多输出10条

## 单位名称匹配验证规则

### 匹配原则
1. **精确匹配**：新闻正文必须明确提到用户输入的单位名称
2. **全称匹配**：如果用户输入简称，同时匹配全称
3. **避免混淆**：同名不同单位需标注区分

### 常见单位名称对照表
| 简称 | 全称 | 匹配关键词 |
|------|------|-----------|
| 绿地建设 | 上海绿地建设集团有限公司 | 绿地建设、绿地建设集团 |
| 中建三局 | 中国建筑第三工程局有限公司 | 中建三局 |
| 中建八局 | 中国建筑第八工程局有限公司 | 中建八局 |
| 中建 | 不明确 | 需确认具体局/公司 |
| 某设计院 | 不明确 | ❌ 无法匹配 |

### 匹配验证流程
```
1. 获取用户输入的单位名称
2. 搜索相关新闻
3. 对每条新闻执行：
   a. 检查新闻正文是否包含单位名称
   b. 如果是简称，检查是否匹配全称
   c. 如果涉及同名单位，标注区分
   d. 无法确认同一单位 → 不收录
4. 输出匹配成功的新闻
```

## 分类定义

### 1. 设计院风暴
**定义**：建筑设计院、规划设计院及相关设计企业的负面经营动态

**子类**：
- **裁员**：大规模裁员、强制裁员、变相裁员（调岗、待岗）
- **降薪**：工资拖欠、绩效取消、薪资腰斩
- **破产**：企业破产清算、重组、倒闭
- **劳动纠纷**：劳动仲裁、违法解除、社保断缴

**典型特征**：
- 涉及主体：建筑设计院、规划设计院、景观设计公司、室内设计公司
- 行为特征：人员优化、组织架构调整、经营困难

### 2. 施工单位困境
**定义**：建筑施工企业及相关从业者的权益受损事件

**子类**：
- **讨薪**：工人集体讨薪、维权事件
- **欠薪**：工资拖欠、工程款纠纷
- **停工**：项目停工、工地停摆
- **破产**：施工企业倒闭、资金链断裂

**典型特征**：
- 涉及主体：施工单位、劳务公司、建筑工人、农民工
- 行为特征：围堵、上访、网络曝光

### 3. 高校专业震荡
**定义**：建筑相关专业在高等教育领域的收缩与调整

**子类**：
- **专业裁撤**：专业被撤销、停止招生
- **招生困难**：报考人数骤降、调剂比例高
- **学生转专业**：集体转专业、转出比例高、转专业潮
- **就业危机**：毕业生就业率下滑、转行率高
- **院系调整**：院系合并、师资缩减

**典型特征**：
- 涉及主体：高校、建筑学院、土木学院、规划学院
- 行为特征：专业停招、招生计划缩减、学生集体转出

### 4. 行业下行信号
**定义**：反映建筑行业整体下行的企业与市场动态

**子类**：
- **企业倒闭**：知名企业破产、退出市场
- **项目停滞**：重大项目停工、烂尾
- **业绩下滑**：营收利润大幅下滑
- **政策影响**：行业政策调整带来的冲击

**典型特征**：
- 涉及主体：房地产企业、建筑集团、设计集团
- 行为特征：债务违约、资产出售、业务收缩

### 5. 从业者维权
**定义**：建筑从业者维护自身权益的集体行动

**子类**：
- **劳动仲裁**：集体仲裁、诉讼维权
- **舆论曝光**：网络发帖、媒体曝光
- **集体行动**：围堵、上访、罢工
- **行业自救**：互助组织、转型分享

**典型特征**：
- 涉及主体：建筑师、规划师、工程师、建筑工人
- 行为特征：维权、曝光、互助

### 6. 职业健康危机
**定义**：建筑从业者因工作导致的健康损害与生命安全事故

**子类**：
- **加班猝死**：设计师、工程师因加班导致的猝死事件
- **过劳伤病**：因长期加班导致的职业病、重症
- **工伤事故**：施工现场安全事故、工伤死亡
- **劳动条件**：超时加班、违法用工、缺乏保障

**典型特征**：
- 涉及主体：建筑师、结构工程师、施工人员、监理人员
- 行为特征：突发疾病、工伤事故、安全责任事故

## 筛选标准

### 模式A筛选标准

**必须包含**：
- 与建筑、规划、土木、房地产设计施工直接相关
- 具有负面性质（裁员、降薪、欠薪、破产等）
- 有明确信息来源
- **发布时间在当前日期前7天内（强制要求）**

**优先选择**：
- **近7天内发布的新闻（强制要求）**
- 有具体数据支撑（裁员人数、欠薪金额等）
- 来自主流媒体或从业者第一手爆料
- 具有行业代表性或区域影响力

### 模式B筛选标准

**必须包含**：
- **新闻正文明确提到用户输入的单位名称（强制要求）**
- 具有负面性质（裁员、降薪、欠薪、破产等）
- 有明确信息来源

**优先选择**：
- 时间较近的新闻
- 有具体数据支撑
- 来源于可验证的平台

**排除项**：
- 无法确认与目标单位相关的新闻
- 同名不同单位的新闻（未明确区分）
- 与建筑行业无关的新闻
- 正面新闻或宣传稿件
- 缺乏来源、纯传言的内容
- **超过7天的旧闻（一律排除，除非有近7天内的后续进展）**
- 广告营销内容

### 时效性强制要求
- **只收录发布时间在当前日期前7天内的新闻**
- **当天为第0天，往前推7天（共8天范围）**
- **示例**：若今天是2025年3月15日，则只收录3月8日至3月15日发布的新闻
- **超过7天的新闻一律不收录，即使事件重要**

### 质量分级
**A级（必选）**：
- 有明确企业/机构名称
- 有具体事件细节
- 有可靠信息来源

**B级（可选）**：
- 有行业传闻但未完全证实
- 有地域但无具体企业名称
- 有事件但缺乏详细数据

**C级（谨慎使用）**：
- 仅有个体从业者爆料
- 缺乏独立信源交叉验证
- 需标注"待证实"

## 多渠道交叉验证

### 验证原则
- **单一信源**：需标注来源，谨慎采用
- **双信源**：两个独立渠道报道，可信度较高
- **多信源**：三个及以上渠道交叉验证，可信度高

### 信源优先级
1. **权威渠道**：官方公告、主流媒体报道
2. **垂直渠道**：行业媒体、专业平台
3. **社群渠道**：从业者爆料、社交平台
4. **投诉渠道**：维权平台、投诉记录

### 标注规范
输出时需标注来源类型：
- `[官方公告]`：政府、司法、企业公告
- `[媒体报道]`：主流新闻媒体
- `[行业平台]`：垂直行业媒体
- `[社交平台]`：微博、知乎、小红书等
- `[投诉平台]`：黑猫投诉、维权记录
